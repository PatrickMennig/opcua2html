opcua2html
====================

##### install 

    $ git clone https://github.com/ja-nz/opcua2html
    $ cd opcua2html/ReactJsApp
    $ npm i
    $ cd -
    $ cd opcua2html/OPCExpressServer
    $ npm i
    $
    $ # start virtual OPC server and express server from repo
    $ # # ja-nz/opcuaServerSide 
    $
    $ # start the dev engine
    $ npm start
