opcua2html
====================

##### install 

    $ git clone https://github.com/ja-nz/opcua2html
    $ cd opcua2html
    $ npm i
    $
    $ # start virtual OPC server in background
    $ node opcua_virtual_arburg_server
    $
    $ # start middleware NodeJS Express server
    $ node node_express_server
    $
    $ # start the dev engine
    $ npm start
