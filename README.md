opcua2html
====================

##### install 

    $ git clone https://github.com/ja-nz/opcua2html
    $ cd opcua2html
    $ npm i
    $
    $ # start test OPC server in background
    $ node opc_server
    $
    $ # start middleware OPC to HTML converting server
    $ node opc_html_middleware
    $
    $ # start the dev engine
    $ npm start
